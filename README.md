# SmartReq-AI

- Turn messy stakeholder input into structured artifacts (user stories, flowcharts, integrations).

## Tech Stack
- Frontend: Next.js (App Router), Tailwind CSS
- Backend: Node.js (Express), Prisma, PostgreSQL
- AI/NLP: Python utilities + LLM orchestration
- Auth, Email, Integrations: Included in backend utilities

- Backend: SmartReq-AI-Backend/
